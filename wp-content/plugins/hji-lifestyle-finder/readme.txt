Lifestyles Finder by Home Junction Inc.

== Description ==

Lifestyles Finder by Home Junction Inc.

Add Lifestyle Finder into posts, pages or text widget with this shortcode:
[lifestyle_finder]

Shortcode has optional parameters: width, height, bookmark.
Width and height supports % and px.

For example:
[lifestyle_finder width="90%" height="600px" bookmark="/schools"]

IMPORTANT: Lifestyle Finder plugin requires HJI Membership plugin to be active.

== Installation ==

1. Unzip hji-lifestyle-finder.zip
2. Upload `hji-lifestyle-finder` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Lifestyle Finder menu will be available in the main Dashboard Navigation.
5. Enter your Home Junction License Key in the Lifestyle Finder settings.
6. Use shortcodes to embed Lifestyles Finder via posts, pages or Text Widget.
